#Rules For Contribution
* name should be dash based for ex: 
```temel-git-egitimi```
* folder structure of project
```
    |-- lectures
        |-- temel-git-egitimi
            |-- images
                |-- .gitkeep
            |-- docs
                |-- .gitkeep
            |-- readme.md
```
* every commit should be a pull request otherwise will not be accepted
* every lecture should have a readme.md file
